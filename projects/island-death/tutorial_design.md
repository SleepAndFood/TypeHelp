## 1. 教程策略

珊瑚湾面向**硬核推理老手**，教程策略为"轻提示、重发现"：
- 每个命令在**首次需要时**才解锁（而非一开始全部暴露）
- 教程文本仅说明"如何使用命令"，不剧透答案
- 6个命令分布在5幕中，与剧情节奏绑定

| 命令 | 幕 | 触发文件 | 标志 | 教程文本 | 必要性 |

## 3. 一次性提示（$first_xxx）触发条件

| 标志 | 触发条件 | 触发位置 | 文本 |
|---|---|---|---|
| `$first_name` | 玩家首次遇到3+角色同时出现的文件 | 09-02-QM-ZM-SM-LM-SU-ZU-6 | "这里有6个人在场。试试 name 查看每个人的详细信息" |
| `$first_note` | 玩家首次发现矛盾证据 | 04-01-PL-1 | "邀请函日期（2月11日）与陈某死亡（2月12日）存在矛盾——委员会在陈某死前就发了招聘。用 note 记录这个疑点" |
| `$first_find` | 玩家文件数超过15个 | 19-04-PL-1 | "你已经收集了15+个文件。用 find 关键词 可以快速查找相关内容" |
| `$first_act` | 玩家进入第四幕 | 22-04-PL-1 | "你已经深入调查。用 act 给当前幕命名，标记你的推理进度" |

## 4. 教程节奏时间轴

```
00段 (meta/教程)
  ├─ 00-readme        → help 解锁, list 解锁
  ├─ 00-invitation    → note 解锁 ($first_note 触发)
  └─ 00-list-of-people → name 解锁

第一幕 ARRIVAL (01-06)
  ├─ 01-01-PL-1       → find 解锁
  └─ 04-01-PL-1       → $first_note 触发

第二幕 THE DROWNING (07-12)
  └─ 09-02-QM-...     → $first_name 触发

第三幕 THE BATH (13-18)
  └─ (无新教程——玩家已熟练)

第四幕 THE NEW BUTLER (19-24)
  ├─ 19-04-PL-1       → $first_find 触发
  └─ 22-04-PL-1       → act 解锁, $first_act 触发

第五幕 RECKONING (25-30)
  └─ (所有命令已解锁——玩家自主使用)
```

## 5. 教程文本样例

### help
```
[@] 欢迎来到珊瑚湾。你是第4任管家。你可以在文本框中输入以下命令：
help（查看命令列表）、list（列出可访问文件）、name（查看人物信息）、note（记录笔记）、find（搜索文件）、act（标记幕名）。
<<set $seen_help to true>>
```

### name
```
[@] 输入 name 可查看各人详细信息。珊瑚湾有22名人员，分属5人委员会、12名员工、3名常客、1名律师和1名管家（你）。
<<set $seen_name to true>>
```

### note
```
[@] 邀请函日期是2月11日——陈某死亡前一天。委员会在陈某死前就已经发出了招聘。这是一个疑点，请记下。
<<set $seen_note to true>>
```

### find
```
[@] 赵某说"因病自杀"，但邀请函上写的是"个人原因离职"。前后矛盾。还有，为什么一艘快艇的引擎被拆走了？用 find 命令可以搜索更多关于快艇的文件。
<<set $seen_find to true>>
```

### act
```
[@] 4任管家，4次死亡。入职→调查→发现→死亡。这是一个模式。用 act 标记当前幕——"猎人与猎物"。
<<set $seen_act to true>>
```
