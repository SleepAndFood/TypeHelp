# Charter：黑匣子 (Black Box)

> 立项文档 · Director Agent 签字

## 1. 题材

**太空船密室推理**——2187 年，载有 6 名船员的火星殖民运输舰"曙光号"在飞行途中，指挥官陈博士于睡眠舱非正常死亡。玩家以"事故调查员"身份，从飞船黑匣子（飞行数据记录器）残存的音频转录文件中还原事件真相。

## 2. 基调

**新本格 + meta 悬疑**。硬科幻设定下，诡计必须物理可行（氧气循环、空气循环、辐射防护等），但允许一层"调查员自己就是事件一部分"的 meta 翻转。

## 3. 时代与世界观

- 时代：2187 年
- 技术：可控核聚变、闭环生态、人工重力、量子通讯
- 飞船："曙光号"，6 名船员，目标火星殖民地"织女站"
- 航程：地球 → 火星，约 7 个月
- 事故时间点：发射后第 89 天

## 4. 核心诡计

**信息差诡计 + 物证诡计**（主+辅）：
- **主**：医生刘博士在 02:45 篡改氧气回收器参数，03:15 指挥官在睡眠舱因舱室氧气浓度骤降死亡。刘博士通过篡改医疗报告，伪装成"自然心脏骤停"。
- **辅**：刘博士的真实动机是向地球黑市倒卖飞船上的稀缺药物，指挥官即将发现此事，刘博士先下手为强。

## 5. 目标受众

推理新手到中级。文件命名清晰、教程节奏舒缓，但仍提供多层验证。

## 6. 目标文件数

**24 个**（worked example 规模，方法论可扩展到 95-100）

- 00 段 meta：7 个（readme / invitation / list-of-names / act-structure / references / dream / final-note）
- 01-26 段正文：17 个

> 备注：完整规模应为 95-100，本 worked example 选用较小规模以演示方法论骨架。

## 7. 幕数

**4 幕**（与 TypeHelp 默认一致）

- 第一幕 IGNORANCE（01-06）——船员发现异常
- 第二幕 CONFUSION（07-13）——第一次死亡与互相猜疑
- 第三幕 TERROR（14-20）——加速崩溃
- 第四幕 RECKONING（21-26）——真相揭示与 meta 收束

## 8. meta 元素方向

**调查员视角 + 嵌套循环**：玩家扮演 2187 年的事故调查员，从 22 世纪的"调查员"读取 89 天前的"船员黑匣子音频"。而 25-WI-K 揭示：调查员本人就是当年事件的"最后一名听众"——ta 选择听完所有音频后，与事件产生精神共鸣，最终在听完最后一个文件后，自己成为事件的"续写者"。
